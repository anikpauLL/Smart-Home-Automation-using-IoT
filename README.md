# Smart-Home-Automation-using-IoT


An **IoT-based Smart Home Automation system** developed as part of **ETE 416** at **Chittagong University of Engineering and Technology (CUET)**. This project demonstrates how household appliances can be controlled wirelessly using an **Arduino UNO**, **Bluetooth communication**, and simple mobile commands.

The system focuses on **cost-effectiveness**, **ease of use**, and **practical applicability**, making it suitable for developing countries and educational purposes.


## Project Overview

Smart Home Automation integrates electronics, communication, and embedded systems to improve convenience, security, and energy efficiency in residential environments. In this project:

* Home appliances are controlled wirelessly via **Bluetooth**
* An **Arduino UNO** acts as the central controller
* A **relay module** safely switches electrical devices
* An **IR sensor** enables basic automation
* A **servo motor** simulates a smart door lock

The system can switch appliances ON/OFF using a mobile phone without complex wiring or internet connectivity.


## Aims & Objectives

* Design a low-cost smart home automation system
* Control home appliances wirelessly via mobile phone
* Demonstrate IoT concepts using Arduino and Bluetooth
* Improve user convenience and energy efficiency
* Provide a scalable foundation for future smart systems


## Working Principle

1. User sends commands from a Bluetooth-enabled smartphone
2. HC-05 Bluetooth module receives commands
3. Arduino UNO processes the commands
4. Relay module switches appliances ON/OFF
5. IR sensor automatically controls lighting based on motion
6. Servo motor operates as a door locking mechanism

## How to Use

1. Power the Arduino-based circuit
2. Pair smartphone with HC-05 Bluetooth module
3. Open Bluetooth controller app
4. Send commands to control bulbs and door lock
5. IR sensor automatically activates lighting on motion detection
